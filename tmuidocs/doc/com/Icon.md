### :seedling: 参数

| 参数名 | 类型 | 默认值 | 描述 |
| :--: | :--: | :--: | :-- |
| label | String | '' | 标签 |
| fontSize | [Number,String] | 34 | 图标大小 |
| color | String | '' | 图标颜色 |
| name | String | '' | 图标名称 |
| spin | [Boolean, String] | true | 是否使图标旋转 |
