##### 开发指南 tmui.design

# 基础页面

***本文档所有出现的vue页面均表示为nvue页面***为了让代码高亮，只能设置代码类型为vue。
```vue
<template>
    <tm-app>
        <!-- 这里是你的页面代码。 -->
    </tm-app>
</template>

<script lang="ts" setup>
import tmApp from "@/tmui/components/tm-app/tm-app.vue"
//你的代码...
</script>

```
