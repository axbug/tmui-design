# 自定义自己的图标


### :tada: 下载图标和字体

- 进入[iconfont](https://www.iconfont.cn/)
- 选好自己的图标，下载图标，勾选好ttf,woff格式，base64
- 下载成功后解压出来，引用自己的图标即可。
- 同时还要记得weex加载你的字体图标，这样可以确保全平台能够使用。weex加载字体图标[点击了解](https://doc.weex.io/zh/docs/modules/dom.html#addrule)

### :tada: 使用图标

引入我的tm-icon图标组件即可使用。
